                     Python SWIG Example Functions

INTRODUCTION

The files in this folder provide examples for accessing SigLib source
files from Python, via the SWIG API. They have been tested with Anaconda
64 bit Python under Windows and Linux.



Please read the appropriate section of the SigLib reference manual for
further information.
