SigLib Machine Learning Example Programs

  

The files in this folder provide examples for the SigLib Artificial Intelligence and Machine Learning functions.

The main SigLib functions implement two-layer convolutional neural networks:
    SDA_TwoLayer2CategoryNetworkFit ()
    SDA_TwoLayer2CategoryNetworkPredict ()
    SDA_TwoLayerNCategoryNetworkFit ()
    SDA_TwoLayerNCategoryNetworkPredict ()

The input data for the training and validation programs is provided in the folder "Original". These files are recorded at 16 kHz but the algorithms are independent of sample rate. The only requirement is that the training and validation data sets are recorded at the same sample rate.

These eexamples are provided for the Microsoft Visual C/C++ and GCC compilers. For graphical result display they use Gnuplot/C, which is included in the SigLib download pachage.

Source Files    Description
preprocess_wav.c    Performs time domain and frequency domain pre-processing of source .wav files
network_train_multi_category.c    Train the multi-category neural network
network_validate_multi_category.c    Validate the multi-category neural network
network_train_single_category.c    Train the single-category neural network
network_validate_single_category.c    Validate the single-category neural network
 
 
Build Scripts 
mbr_multi_category.bat    Batch file to build and run multi category training and validation, with Microsoft Visual C/C++ compiler
mbr_multi_category_withDataAugmentation.bat    Batch file to build and run multi category training and validation, with additional data augmentation, with Microsoft Visual C/C++ compiler
mbr_single_category.bat    Batch file to build and run single category training and validation, with Microsoft Visual C/C++ compiler
mbr_single_category_with_scatter_diagram.bat    Batch file to build and run single category training and validation, with additional scatter diagram, with Microsoft Visual C/C++ compiler
gbr_multi_category.sh    Shell script to build and run multi category training and validation, with GCC
gbr_multi_category_withDataAugmentation.sh    Shell script to build and run multi category training and validation, with additional data augmentation, with GCC
gbr_all_single_category.sh    Shell script to build and run single category training and validation, with GCC
gbr_all_single_with_scatter_diagram.sh    Shell script to build and run single category training and validation, with additional scatter diagram, with GCC
 
Motor Vibration Demonstration Scripts
mbr_single_category_motorMode.bat    Batch file to build and run the training and validation, with Microsoft Visual C/C++ compiler
gbr_single_category_motorMode.sh    Shell script to build and run the training and validation, with GCC
