SigLib Example Programs

  

This directory contains the ANSI C source example programs for SigLib.

All of these examples can be compiled with the free Microsoft Visual Studio and the GCC (Cygwin, Linux or OS-X) C/C++ compilers using Gnuplot/C which is available from : http://www.numerix-dsp.com/files.
After compiling the examples they can all be executed using "runall.bat".

This folder includes some batch files included to assist with the recompilation of these examples under DOS/Windows :

mb.bat - Build an example using the Microsoft compiler
mbr.bat - Build and run an example using the Microsoft compiler
mbuildall.bat - Build all the examples using the Microsoft compiler
mb_64.bat - Build an example using the Microsoft 64 bit compiler
mbr_64.bat - Build and run an example using the Microsoft 64 bit compiler
mbuildall_64.bat - Build all the examples using the Microsoft 64 bit compiler
gb.bat - Build an example using the Gnu compiler
gbr.bat - Build and run an example using the Gnu compiler
gbuildall.bat - Build all the examples using the Gnu compiler
runall.bat - Execute all of the examples in alphabetical order

There are three makefiles included to assist with the recompilation of these examples under Linux :

makefile_mat - compile the matrix example
makefile_sig - compile the sig_gen example
makefile_wav - compile the dtmfwav example
