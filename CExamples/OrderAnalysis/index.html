SigLib Order Analysis Example Programs

  

The files in this folder provide examples of how to use the order analysis functions.

The input data sequence is taken from the files "Const.dat" "Variable.dat" this data is in 2 columns containing magnitude and machine speed (RPM).

The batch files supplied work with the GCC and Microsoft Visual Studio compiler (32 and 64 bit versions supplied) however the example programs can be built using almost any 32 or 64 bit compiler.

Filename    Description
order.c    Source file for generating an order plot.
waterfall.c    Source file for generating a waterfall plot.
Const.dat    Constant speed data file
Variable.dat
Variable Speed data file



GCC Batch files for building and running test functions
gb.bat
Batch file to build source file with SigLib using Microsoft Compiler
gbrocd.bat
Order plots for constant speed data - dB scaling
gbrocl.bat
Order plots for constant speed data - Linear scaling
gbroVds.bat
Order plots for variable speed data - dB scaling against speed
gbroVdt.bat
Order plots for variable speed data - dB scaling against time
gbroVS.bat
Spectrogram plot for variable speed data - dB scaling
